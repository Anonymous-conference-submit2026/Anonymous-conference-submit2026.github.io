üîç3. Key Observations on Heterogeneity

Pruning attention layers and activation function layers jointly is non‚Äëtrivial because these two components behave very differently during backpropagation and fine‚Äëtuning. Conventional pruning metrics‚Äîwhether gradient‚Äëbased or handcrafted‚Äîoften fail to capture this cross‚Äëlayer heterogeneity. We summarize two key observations that reveal the root difficulty of joint depth pruning in ViTs.

‚Ä¢ Observation 1: Gradient Disparity

Attention layers and activation function layers exhibit large differences in gradient magnitude during backpropagation. When importance scores are learned through training, activation layers consistently show much higher gradients than attention layers, leading to biased importance estimation if gradient‚Äëbased metrics are used. This gradient gap persists even after multiple epochs.

Gradient Disparity Between Layer Types

Challenge: Gradient‚Äëbased pruning metrics or methods fail because attention layers are systematically undervalued and therefore incorrectly pruned.

‚Ä¢ Observation 2: Recovery Asymmetry

The two layer types also show drastically different accuracy recovery behaviors. Pruning attention layers causes mild initial accuracy drops but requires long fine‚Äëtuning to recover. In contrast, pruning activation function layers triggers severe accuracy collapse (often over 90%) but recovers very quickly during fine‚Äëtuning. After 10 epochs of recovery, both layer types return to similar accuracy levels.

Recovery Asymmetry After Pruning

Challenge: Handcrafted or single‚Äëstep pruning metrics cannot model this asymmetric behavior and therefore make suboptimal joint pruning decisions.

These two heterogeneity observations highlight why joint depth pruning is difficult and directly motivate the design of BoundaryDPT, which addresses both challenges through separate pruning statistic collection and training‚Äëbased layer selection.

üß©4. Method: BoundaryDPT

BoundaryDPT is a three‚Äëstage pruning framework designed to jointly prune attention layers and activation function layers in Vision Transformers. Unlike conventional depth pruning, which often collapses accuracy, BoundaryDPT resolves key heterogeneity issues‚Äîgradient disparity and recovery asymmetry‚Äîwhile ensuring the pruned model remains structurally valid and computationally efficient.

‚Ä¢ Stage 1: Identifying Redundant Layers

The first stage determines how many attention layers and activation layers should be pruned. To achieve this, BoundaryDPT uses a lightweight Model Accuracy Predictor (MAP) that estimates the accuracy of different pruning configurations. By collecting data through a fast iterative prune‚Äëand‚Äëfinetune procedure and fitting a polynomial predictor, MAP efficiently searches the pruning space.

After determining pruning ratios, BoundaryDPT performs a training‚Äëbased layer importance evaluation. Each candidate layer receives a learnable importance parameter, which is refined through backpropagation. Layers with the lowest importance within each layer type are removed. Crucially, attention and activation layers are evaluated separately‚Äîavoiding biased comparisons caused by their heterogeneous gradient scales.

‚Ä¢ Stage 2: Removing Layers and Fine‚Äëtuning

Once the least important layers are removed, the pruned ViT is fine‚Äëtuned to recover accuracy. BoundaryDPT optionally supports self‚Äëdistillation, where the original full model acts as a teacher. This boosts recovery without introducing external models.

‚Ä¢ Stage 3: MLP Merging for Structural Simplification

With activation layers removed, adjacent linear layers in MLP blocks can be safely merged. This operation reduces the model‚Äôs depth and delivers additional inference speedups without affecting accuracy. The result is a structurally compact and efficient ViT.

‚Ä¢ BoundaryDPT+: Depth + Width Pruning

BoundaryDPT can be further combined with existing width pruning algorithms to yield BoundaryDPT+, a unified framework for extreme ViT compression. This integration significantly expands the achievable speed‚Äìaccuracy Pareto frontier.
